    grid settings

    variables

    min-base-font-size

    $min-base-font-size: 16px !default;

    Description

    Defines the base font size used for the project (should be left at 16px)

    min-gutter-outer

    $min-gutter-outer: true !default;

    Description

    Sets outer gutter (true/false)

    min-gutter

    $min-gutter: 32px !default;

    Description

    Sets standard gutter size (px)

    min-gutter-small

    $min-gutter-small: 16px !default;

    Description

    Sets small varient gutter size (px)

    min-wrapper-width

    $min-wrapper-width: 75em !default;

    Description

    Sets standard wrapper width (recommend em)

    min-wrapper-wide-width

    $min-wrapper-wide-width: 90em !default;

    Description

    Sets wide varient wrapper width (recommend em)

    min-breakpoint-generators

    $min-breakpoint-generators: false !default;

    Description

    Defines additional breakpoints to be generated by passing a breakpoint name and a max-width.

    Example

    $min-breakpoint-generators: l 920px, m 767px, s 420px;
